法律网站启动说明
================

如果网站显示空白，请按以下步骤操作：

1. 确保已安装 Node.js
   - 访问 https://nodejs.org/ 下载并安装
   - 安装完成后，打开命令行输入 node -v 和 npm -v 检查是否安装成功

2. 安装项目依赖
   在项目目录下打开命令行（PowerShell 或 CMD），运行：
   npm install

3. 启动开发服务器
   运行：
   npm run dev

4. 访问网站
   运行 npm run dev 后，命令行会显示服务器地址，通常是：
   - http://localhost:3000
   - 或者 http://localhost:5173（如果3000端口被占用）
   
   请查看命令行输出的实际地址，然后访问该地址

常见问题：
- 如果 npm install 失败，尝试使用国内镜像：
  npm install --registry=https://registry.npmmirror.com

- 如果端口 3000 被占用，Vite 会自动使用其他端口，查看命令行输出

- 如果仍然空白，按 F12 打开浏览器开发者工具，查看 Console 标签页的错误信息

关于网站部署：
================

npm run dev 启动的是本地开发服务器，只能在本机访问（http://localhost:3000）。

要让其他人通过网址访问您的网站，需要将网站部署到互联网服务器上。

详细部署方法请查看：部署说明.md

快速部署方法（使用 Vercel）：
1. 安装：npm install -g vercel
2. 在项目目录运行：vercel
3. 按照提示操作，即可获得一个免费的网址
