# 法适天下 - 开源法律服务平台

**法适天下**是一个完全开源的法律服务平台，致力于为广大用户提供免费、便捷的法律服务。我们欢迎所有对法律感兴趣的朋友加入我们的开源社区，共同完善这个平台。

## 🌟 项目特色

- **完全开源** - 代码透明，欢迎社区贡献和协作开发
- **免费使用** - 绝大部分功能免费，让法律知识普及到每个人
- **社区驱动** - 由社区共同维护和完善
- **功能丰富** - 提供常见答疑、精选案例、法律文书模板和法律工具等功能

## 功能特性

### 📋 常见答疑
- 分类展示各类法律问题
- 可展开/折叠的问题解答
- 涵盖劳动法、合同法、婚姻法、侵权责任法等多个领域

### 📚 精选案例
- 真实案例解析
- 按类别筛选案例
- 详细的案例分析和法律要点

### 📝 法律文书模板
- 多种常用法律文书模板
- 支持在线查看和复制
- 包括劳动合同、租赁合同、离婚协议等

### 🛠️ 法律工具
- 加班费计算器
- 诉讼时效计算器
- 违约金计算器
- 利息计算器

## 技术栈

- React 18
- React Router 6
- Vite
- CSS3

## 安装和运行

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 页面说明

- **首页** (`/`): 展示平台介绍和核心功能入口
- **常见答疑** (`/faq`): 法律问题解答
- **精选案例** (`/cases`): 真实案例解析
- **法律文书模板** (`/templates`): 各类法律文书模板
- **法律工具** (`/tools`): 实用的法律计算工具
